# Sea War Python

海战模拟游戏系统，基于Python后端逻辑和Unreal Engine前端显示。

## 项目概述

Sea War是一个海战模拟游戏系统，采用Python实现游戏逻辑，Unreal Engine实现3D显示。系统支持单机模式和多人联网模式，采用帧同步技术实现多客户端之间的数据一致性。

### 主要特性

- 基于组件的实体系统
- 对象池技术优化性能
- 帧同步网络模型
- Python与UE的TCP通信
- 支持AI和人类玩家
- 3D战场可视化

## 安装与运行

### 环境要求

- Python 3.8+
- Unreal Engine 5.0+
- 依赖库：见requirements.txt
